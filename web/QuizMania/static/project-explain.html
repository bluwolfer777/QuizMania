Cosa è QuizMania?


QuizMania è un progetto ispirato alla famosa piattaforma di apprendimento basata
sui quiz Kahoot, ideata da cinque studenti della CPT di Trevano perché impossibilitati
ad andare in settimana bianca.
Questo progetto è stato sviluppato nell'arco di cinque giorni usufruendo dell'aiuto dell'intelligenza
artificiale per aiutarci con la parte grafica e utilizzando cinque linguaggi di programmazione differenti:
HTML
CSS
JavaScript
SQL
Python + Flask
